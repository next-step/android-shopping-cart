# android-shopping-cart

## 🚀 1단계 - 장바구니(상품 목록)

### 요구 사항

- ViewModel, Hilt 등은 장바구니 미션에서 활용하지 않는다. 컴포즈 학습에 집중하자.
- 상품 목록 화면을 구현할 때 Lazy 컴포넌트를 활용한다.
- 컴포저블 함수가 너무 많은 일을 하지 않도록 분리하기 위해 노력해 본다.
- 의미있는 단위의 함수를 모아 별도의 파일로 분리해본다.

### 개발 사항

- [ ] 상단바 컴포넌트 생성
- [ ] 상품 아이템 컴포넌트 및 상품 데이터 생성
- [ ] 상품 리스트 컴포넌트 생성
