# android-shopping-cart
### Step1. 장바구니 (상품 목록)
- 프로젝트 첫 코드 정리 및 상품 목록 화면 컴포넌트 추가
- 타이틀 컴포넌트 구현
- 우측 상단 아이콘 구현
- 상품 목록 아이템 컴포넌트 구현

### Step2. 장바구니 (상품 상세)
- Navigation 추가
- Navigation 코드 추가 및 상세 화면, 장바구니 화면 껍데기 추가
- 상품 상세 화면 구현
- 장바구니 빈 화면 구현

### Step3. 장바구니 (담기)
- 장바구니에 담을 수 있는 CartItem List 구현
- CartItem에 해당하는 아이템 구현
- 테스트 코드 작성

### Step4. 장바구니 (수량)
- 상품 목록에서 상품의 수량을 조절할 수 있도록 구현 
- 테스트 확인
